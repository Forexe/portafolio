Bootstrap

Cree proyectos responsivos y móviles en la web con la biblioteca de componentes front-end más popular del mundo.

Bootstrap es un kit de herramientas de código abierto para desarrollar con HTML, CSS y JS. Crea rápidamente prototipos de tus ideas o construye toda tu aplicación con nuestras variables y mixins Sass, sistema de cuadrícula responsivo, extensos componentes preconstruidos y potentes plugins basados en jQuery.

Instalación

Incluya los archivos Sass y JavaScript de origen de Bootstrap a través de npm, Composer o Meteor. Las instalaciones administradas por paquetes no incluyen documentación, pero sí nuestro sistema de compilación y el archivo Léame.
